визуально состоит из двух частей


1.  горизонтальная "линейка" с временной шкалой

	"линейка" показывает динамику количества материалов от даты.
	есть 3 масштаба Месяц, Неделя, День.

"scrollEvents.update":

"header" - обновляет положение хедера таймлайна - прибивая его при скролле position:fixed к верху страницы

методы -

"init" - инициализация. определение массивов для данных.

"load" - загружает новые даты для навигатора -  при загрузке всех выставляется флаг full_load

"plotInit" - рисует сам график время-количество статей с помощью flot-плагина
